# Grimoire

Applicazione React statica per consultazione contenuti e wiki tematica, con una `sheet` lato client in sviluppo.

## Comandi principali

```bash
npm start
npm run build
npm test -- --watchAll=false
npx playwright test tests/example.spec.js --project=chromium
npx playwright test tests/table-detail-pages.spec.js --project=chromium
```

## Deploy

```bash
npm run deploy
```
